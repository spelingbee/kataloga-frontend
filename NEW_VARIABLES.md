# New Variables

This file documents the new variables that have been added to the project. These variables are placeholders and will require proper implementation.

## `app/pages/orders/repeat.vue`

*   `recentOrders`: This variable is a `ref` that is initialized with mock data. It should be replaced with a call to the `order` store to fetch the recent orders.
*   `favoriteCombos`: This variable is a `ref` that is initialized with mock data. It should be replaced with a call to the `order` store to fetch the favorite combos.

## `app/pages/select-restaurant.vue`

*   `error`: This variable is a `ref` that is initialized to `null`. It should be used to store any errors that occur when selecting a restaurant.
